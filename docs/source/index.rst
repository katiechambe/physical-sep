Welcome to the documentation for **Physical Sep**!
===================================

**Physical Sep** is a Python package created to predict the physical separation of galaxies with observed (projected) position-velocity meansurements. The prediction model is trained on data from the IllustrisTNG simulation. 

Check out the :doc:`usage` section for further information.


.. note::

   This project is under active development.
